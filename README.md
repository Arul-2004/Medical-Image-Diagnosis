Medical Image Diagnosis is a deep learning-based system that automatically classifies X-ray and MRI scans to detect diseases such as pneumonia, tumors, or fractures. Using CNNs and advanced transfer learning models like ResNet or EfficientNet, the system achieves high accuracy and provides visual explanations with Grad-CAM. Features like GAN-based data augmentation, ensemble models, and integration of patient metadata help improve reliability, making it a useful tool for faster and more informed medical decision-making.
